# Trader Performance vs Market Sentiment Analysis

## Objective
Analyze how market sentiment (Fear & Greed) affects trader behavior and performance.

## Methodology
- Data cleaning and preprocessing
- Date conversion and merging
- Feature engineering
- Sentiment-based performance analysis
- Trader segmentation

## Key Insights
- Extreme Greed showed highest profitability
- Fear periods had highest activity and volatility
- Frequent traders contributed most to market activity

## Strategy Recommendations
- Reduce exposure during Fear periods
- Increase participation during Extreme Greed

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
